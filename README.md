Welcome to my first-ever Data Analytics project!

I built this to answer a simple question: **What actually makes a good student?** Is it just about grinding hours of study, or do things like sleep, social media, and mental health matter just as much?

To figure this out, I took a dataset of student habits and used **Python (Pandas, Matplotlib, and Seaborn)** to dig into the numbers.

### What I Built

Instead of generating a dozen separate charts, I challenged myself to build a **single, unified Dashboard**. It gives a clear, bird's-eye view of how lifestyle choices correlate with final exam scores.

Here is what the analysis covers:
- **The Big Picture:** A correlation heatmap to see exactly which habits are tied to good (or bad) grades.
- **Study vs. Grades:** A regression plot to check the direct impact of study hours.
- **The Screen Time Trap:** Boxplots comparing exam scores across different levels of social media usage.
- **Health & Mind:** Visualizing how diet quality and mental health ratings affect academic success.

## Want to run it yourself?

If you want to poke around the code or run the analysis on your machine, it's super easy to set up.

1. Make sure you have Python installed.
2. Install the required libraries:
   ```bash
   py -m pip install pandas matplotlib seaborn numpy
   ```
3. Clone this repo, open the folder in your terminal, and run the script:
      ```bash
   py [project-name].py
   ```
